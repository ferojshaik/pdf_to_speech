PDF to Speech APK Builder

Method 1: Use APK Builder Online

1. Go to Appetize.io

2. Upload your Python code

3. They'll build the APK for you

Method 2: Use Termux (Easiest)

1. Install Termux on your Android phone

2. Run these commands in Termux:

pkg update && pkg upgrade
pkg install python
pip install pdfminer.six pyttsx3
# Then run your Python script directly

Method 3: Use Pydroid 3

1. Install Pydroid 3 from Play Store

2. Copy your Python code

3. Install required packages in Pydroid 3

4. Run your script directly

Method 4: Convert to APK with Auto-py-to-exe

1. Install: pip install auto-py-to-exe

2. Run: auto-py-to-exe

3. Select your main.py file

4. Choose "One File" and "Window Based"

5. Click "Convert"
